# Tech Stack - Clever Data Project

## Introduction
This document defines the core technologies and libraries used in the Clever Data Project for interacting with the Canvas LMS API.

## Core Technologies
- **Programming Language:** Python 3.x
- **HTTP Client:** `requests` - Used for all API requests to the Canvas LMS.
- **Data Serialization:** `json` - Used for parsing responses and consolidating data for output.

## Integrations
- **API Source:** Canvas LMS API (RESTful endpoint)
- **District Context:** Initially focused on Broward Schools, but designed for district-agnostic API interactions.

## Development Environment
- **IDE/Project:** Spyder (Inferred from `.spyproject` configuration)
- **Version Control:** Git
